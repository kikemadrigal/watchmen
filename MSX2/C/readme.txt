# Instalación

1. Descárgate las dependias (o programas que necesita fusion C para funcionar) del siguiente enlace:

http://msx.tipolisto.es/files/fusioc-tools.zip

pon la carptea tools con todo su contenido en la raiz.

2. Cambia el nombre para tu juego en el archivo principal y dentro de compil.bat (si estás en windows) o dentro del Makefile (si estas en linux).

3. Pon en el cmd de Windows "make.bat" o en un terminal de linux (make) y pulsa intro.
